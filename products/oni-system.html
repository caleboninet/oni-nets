Full-Size Competition System

ONE PLATFORM. TWO ENVIRONMENTS.

The Oni System is built around the idea that serious volleyball players shouldn't need to purchase a completely different system every time the court changes. The core system stays the same. The base configuration adapts.

Configuration 01
Grass Configuration

Designed for outdoor grass play. The base component is suited for grass court environments where players compete outdoors.

Configuration 02
Indoor Configuration

Designed for indoor court play. The same system transitions to an indoor base configuration without replacing the core components.

Final product specifications and configurations are still being completed ahead of production.

COMMON QUESTIONS

When will the Oni System be available?

The first production run is currently being prepared. Join early access to receive updates, first-release pricing information, and priority notification when the system ships.

How does the modular base system work?

The Oni System is designed so the core components remain the same while the base or bottom configuration can be changed for grass or indoor use. Final configuration details will be shared ahead of production.

Can I use the same system for both grass and indoor?

Yes. That is the core concept behind the Oni System. Rather than purchasing separate systems for each environment, the Oni System is designed around interchangeable base components to support both.

What is the price?

Pricing will be confirmed and communicated to early-access members ahead of the first production release. Join early access to receive first-release pricing.

Is this system suitable for competitive play?

The Oni System is being developed for athletes who take the sport seriously—players who train, travel, and compete. Development and testing are ongoing ahead of the first production run.
